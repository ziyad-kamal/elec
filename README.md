# موقع Portfolio متحرك - مصمم جرافيك احترافي 🎨

موقع ويب فريد وعصري لعرض أعمال مصمم جرافيك مع تأثيرات حركية رائعة وتصميم استجابي.

## 📋 المميزات الرئيسية

### ✨ الرسوميات والحركات

- **Smooth Animations**: تأثيرات حركية سلسة على جميع العناصر
- **Floating Cards**: بطاقات عائمة متحركة في قسم Hero
- **Parallax Scrolling**: تأثير منظور عند التمرير
- **Hover Effects**: تأثيرات عند الوقوف على العناصر
- **Counter Animations**: عدادات متحركة في قسم الإحصائيات

### 🎯 الأقسام الرئيسية

1. **Navigation Bar**
    - شريط تنقل عصري مع تأثيرات hover
    - قائمة جوال responsive
    - مؤشر للصفحة الحالية

2. **Hero Section**
    - عنوان ديناميكي مع animations
    - زر تحريك للأسفل
    - بطاقات عائمة متحركة
    - تأثيرات parallax

3. **Portfolio Gallery**
    - معرض 8 أعمال متنوعة
    - نظام فلاترة تفاعلي (الكل، هوية بصرية، واجهات، طباعة، وسائل اجتماعية)
    - تأثيرات حركية عند الفلترة
    - بطاقات responsive تتكيف مع حجم الشاشة

4. **About Section**
    - معلومات شخصية
    - قائمة المهارات
    - إحصائيات متحركة (250+ مشروع، 80+ عميل، إلخ)

5. **Contact Section**
    - نموذج تواصل تفاعلي
    - معلومات الاتصال (بريد، هاتف، موقع)
    - رسائل نجاح/خطأ مخصصة

6. **Footer**
    - روابط سريعة
    - وسائل التواصل الاجتماعي
    - معلومات الحقوق

## 🛠️ التكنولوجيات المستخدمة

- **HTML5**: هيكل الصفحة
- **CSS3**: التصميم والرسوميات والحركات
    - CSS Grid و Flexbox للتخطيط
    - Gradients للألوان المتدرجة
    - Animations و Transitions للحركات
    - Media Queries للاستجابة
- **JavaScript Native**: التفاعليات والديناميكية
    - Smooth Scrolling
    - Event Listeners
    - Intersection Observer API
    - DOM Manipulation

## 📱 التوافقية والاستجابة

✅ **Fully Responsive**

- سطح المكتب (1200px+)
- أجهزة لوحية (768px - 1199px)
- الهواتف الذكية (480px - 767px)
- الهواتف الصغيرة (أقل من 480px)

## 🚀 كيفية الاستخدام

### خطوات سهلة:

1. **استخرج الملفات الثلاثة**:
    - `index.html` - هيكل الصفحة
    - `styles.css` - التصميم والرسوميات
    - `script.js` - التفاعليات

2. **افتح الملف**:
    - انقر نقراً مزدوجاً على `index.html`
    - أو افتح من متصفحك

3. **لا توجد متطلبات إضافية** - يعمل مباشرة بدون backend!

## 📝 التخصيص

### تغيير المعلومات الشخصية:

في `index.html`:

- غيّر "ديزاين ستوديو" باسمك
- عدّل البريد الإلكتروني والهاتف
- عدّل المعلومات في قسم "حولي"

### إضافة أعمالك:

في قسم `Portfolio`:

```html
<div
    class="portfolio-item"
    data-category="branding"
>
    <div class="portfolio-image">
        <div
            class="placeholder"
            style="background: linear-gradient(...);"
        >
            <!-- يمكنك وضع صورة هنا بدل الـ placeholder -->
        </div>
    </div>
    <div class="portfolio-info">
        <h3>اسم المشروع</h3>
        <p>وصف المشروع</p>
        <span class="category-badge">الفئة</span>
    </div>
</div>
```

### تغيير الألوان:

في `styles.css`:

```css
:root {
    --primary-color: #667eea; /* لونك الأساسي */
    --secondary-color: #764ba2; /* اللون الثانوي */
    --accent-color: #f5576c; /* لون التمييز */
    /* ... إلخ */
}
```

## 🎬 الحركات الرئيسية

| الحركة         | الموقع           | الوصف             |
| -------------- | ---------------- | ----------------- |
| `fadeIn`       | عنوان Hero       | ظهور سلس للعنوان  |
| `slideInRight` | نص Hero          | دخول من اليسار    |
| `float`        | البطاقات العائمة | حركة طائرة عمودية |
| `pulse`        | خلفية Hero       | نبض الخلفية       |
| `bounce`       | مؤشر التمرير     | قفزة متكررة       |
| `fadeInUp`     | بطاقات المعرض    | ظهور من الأسفل    |
| `spin`         | أيقونات البطاقات | دوران مستمر       |
